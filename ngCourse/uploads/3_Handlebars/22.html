Conditional Expressions
Conditional Expressions

Render code only if a condition is
fulfulled

Using
{{#if condition}} {{/if}}
or
{{unless condition}} {{/unless}}
<h1>Posts</h1>
<ul class="posts">
{{#posts}}
<li>
<strong>
<a href="#/posts/{{id}}">{{title}}</a>
</strong>
{{#if author}}
<span class="author">by
{{author}}</span>
{{/if}}
</li>
{{/posts}}
</ul>
If author is
false-like
value,
this code will
not be rendered
