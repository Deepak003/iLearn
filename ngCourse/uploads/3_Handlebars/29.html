Registering Custom
Helpers

Handlebars.js supports extension
of the
built-in helpers

Using
Handlebars.registerHelper(…)
:
Handlebars.registerHelper ('tags', function(tags){
var html = '<ul class="tags-list">',
tag;
for(tag in tags){
html += '<li class="tag">' + tag + '</li>';
}
return
new Handlebars.SafeString(html)
;
});
SageString is
mandatory here
Otherwise the
result will be
escaped
