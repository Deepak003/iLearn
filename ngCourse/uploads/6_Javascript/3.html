Why use client-side
programming?
PHP already allows us to create dynamic
web pages. Why also use client-side
scripting?

client-side scripting (JavaScript) beneﬁts:

usability
: can modify a page without
having to post back to the server (faster UI)

eﬃciency
: can make small, quick changes
to page without waiting for server

event-driven
: can respond to user actions
like clicks and key presses
